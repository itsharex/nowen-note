1. 为表格 Block 身份补失败测试，再将 `table` 接入编辑器、转换器和后端 Block 索引。
2. 为前后端安全节点归一化补失败测试，再实现对称的表格节点白名单、属性校验、容量限制和嵌套 ID 校验。

4. 为后端应用与路由补冲突、非顶层、回滚、单次版本递增和完整索引同步用例，再收紧应用边界。
5. 更新 Block Patch 协议文档，运行前后端定向测试、类型检查和前端生产构建。
